# Panorama Scholarly Group (PSG)

Panorama Scholarly Group (PSG) is an independent open-access academic publishing initiative
dedicated to promoting transparent, ethical, and accessible scholarly communication.

This repository hosts the source files for the official publisher website of Panorama Scholarly Group,
built and deployed using GitHub Pages. The website provides public information on the publisherâ€™s
mission, governance structure, journals, and publishing policies.

## Scope

The publisher supports peer-reviewed academic journals across the humanities,
social sciences, and interdisciplinary research fields. Each journal operates under
an independent editorial board and follows internationally recognized standards
of peer review and publication ethics.
